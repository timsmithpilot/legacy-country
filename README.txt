LEGACY — ONE-PAGE WEBSITE

1. Unzip this folder.
2. Copy all files into your GitHub repository folder.
3. Open index.html with Live Server to preview.
4. In index.html, replace YOUR-EMAIL-HERE with the booking email address.
5. Commit and push the files to GitHub. Cloudflare Pages should update automatically.

Included:
- The mural hero photo
- A cream transparent version of the supplied black Legacy logo
- The five member portraits
- Embedded YouTube video
- Mockup-style navigation buttons
- Meet the Band pop-up with photos and bios
- Responsive phone and desktop styling

The site uses Google Fonts when an internet connection is available. It falls back to normal system fonts offline.
